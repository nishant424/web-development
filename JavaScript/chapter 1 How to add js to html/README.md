# ways to add JavaScript to your html file 

## internal JavaScript:
```html
<head>
    <! –– inside head ––>		
    <script language="JavaScript" type="text/JavaScript">
            // JavaScript code
    </script>
</head>
<body>

    <! –– inside body ––>
    <script>
		/* JavaScript code */
    </script>
</body>
```

<br>

## external JavaScript:
```html
<head>
    <script src="script.js" language="JavaScript" type="text/javaScript">
    </script>
</head>
```
