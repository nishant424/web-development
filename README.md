## Folder Structure of a Web-development Project

```js
PROJECT_ROOT
|----- Pages        # page files
|      '--- api     # serverless functions
|----- Styles       # StyleSheets
|----- Script       # Scripts
|----- media        # contains all important media
|----- public       # Static Assets

```
